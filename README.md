# README

* Ruby Version 2.6.3
* Rails Version 5.2.4

# Step to start project

* First Clone the project 
* Run `bundle install`
* Run `rails db:create && rails db:migrate && rails db:seed`
* Run Server `rails s`
* Hit [locahost]`http://localhost:3000` on your browser

# Step Run Test Cases

* run `rsepc`
